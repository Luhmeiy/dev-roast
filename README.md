# DevRoast

Drop your code, get roasted. A fun way to see how bad your code really is.

## What It Does

- **Paste your code** and get a brutally honest score
- **Roast Mode** - toggle for extra harsh feedback
- **Leaderboard** - see the worst code ranked by shame
- **Syntax highlighting** - beautiful code display

## Features

- Code input with line numbers
- Real-time score calculation
- Share your roast results
- Check the leaderboard to see if your code is the worst

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
